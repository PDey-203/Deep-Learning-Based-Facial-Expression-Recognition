<h1 align="center" id="title">Deep-Learning-Based-Facial-Expression-Recognition</h1>

<p id="description">Implemented transfer learning with EfficientNetB0 for facial emotion recognition on the FER dataset achieving 65 percent accuracy while handling class imbalance using weighted training. Built a real-time webcam-based system for live facial expression detection and inference.</p>

  
  
<h2>🧐 Features</h2>

Here're some of the project's best features:

*   Developed a facial emotion recognition system using EfficientNetB0 transfer learning on the FER dataset.
*   Built a real-time facial expression detection pipeline using OpenCV Haar Cascades for face detection.
*   Achieved 65% emotion classification accuracy across multiple facial expression categories.
*   Implemented class-imbalance handling with weighted training to improve classification reliability.

  
  
<h2>💻 Built with</h2>

Technologies used in the project:

*   Python
*   TensorFlow/Keras
*   OpenCV (Haar Cascades)
